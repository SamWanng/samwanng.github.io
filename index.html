NSDictionary 字典

NSDictionary 不可变字典

  • initWithObjectsAndKeys 初始化一个字典对象 > 在初始化时 一次性存储多个键值对(条目) value 在前 key 在后 之间用 , 隔开
NSDictionary *jinPing = [[NSDictionary alloc] initWithObjectsAndKeys:@"jinPing",@"name",@"男",@"sex",@14,@"aage",nil];
  • valueForKey 通过 key 获取 value
NSString *sex = [jinPing valueForKey:@"sex"];
  • allKeys 获取所有的 key
NSArray *allkey = [jinPing allKeys];
  • allValue 获取所有的 value
NSSArray *allvalue = [jinPing allValues];

遍历字典

for(int i = 0; i < allkey.count; i ++) {
    //获取对应的 key
    NSString *key = [allkey objectAtIndex:i];
    //通过对应的 key 获取对应的 value
    NSString *value = [jinPing valueForKey:key];

快速枚举

for (NSString *key in jinPing) {
    //通过 key 快速获取对应的 value
    NSLog(@"value = %@",[jinPing valueForKey:key]);

NSMutableDictionary 可变字典

NSMutableDictionary *daHa = [[NSMutableDictionary alloc]initWithObjectsAndKeys:@"daHa",@"name",@"m",@"sex",@"78",@"age",@"zuYing",@"hobby",nil];

添加 / 修改 / 删除

[daHa setObject:nil forKey:@"weight"];

[daHa setValue:nil forKey:@"weight"];

setObject:forKey: 和 setValue:forKey: 都可以添加键值对 但是,前者不允许添加的为空
setValue:forkey: 除了可以添加键值对也可以修改,甚至产生删除效果.如果键值对存在则修改 value 的值, 如果不存在则会添加新的键值对. 如果给的 value 参数为 nil, 则删除对应的键值对

  • removeObjectForKey 同过给定的 key 删除相应的键值对
[daHa removeObjectForKey:@"sex"];
  • removeAllObjects 删除所有键值对
[daHa removeAllObjects];

使用 字典 \ 数组 来表示 省 市 区

//海淀区 和 朝阳区
NSDictionary *haiDian = [NSDictionary dictionaryWithObjectsAndKeys:@"海淀区",@"areName",@211,@"population", nil];
        
NSDictionary *chaoYang = [NSDictionary dictionaryWithObjectsAndKeys:@"朝阳区",@"areName",@183,@"population", nil];
        
//区数组
NSArray *ares1 = [NSArray arrayWithObjects:haiDian,chaoYang, nil];
        
//北京市
NSDictionary *beiJing = [NSDictionary dictionaryWithObjectsAndKeys:@"北京市",@"cityName",ares1,@"areArr", nil];
        
//二七区 和 金水区
NSDictionary *erQi = [NSDictionary dictionaryWithObjectsAndKeys:@"二七区",@"areName",@54,@"population", nil];
        
NSDictionary *jinShui = [NSDictionary dictionaryWithObjectsAndKeys:@"金水区",@"areName",@81,@"population", nil];
        
NSArray *ares2 = [NSArray arrayWithObjects:erQi,jinShui, nil];
        
//郑州市
NSDictionary *zhengZhou = [NSDictionary dictionaryWithObjectsAndKeys:@"郑州市",@"cityName",ares2,@"areArr", nil];
        
//城市数组
NSArray *citys = [NSArray arrayWithObjects:beiJing,zhengZhou, nil];
        
//创建省
NSDictionary *provinceDic = [NSDictionary dictionaryWithObjectsAndKeys:@"河南省",@"provinceName",citys,@"cityArr", nil];
        
NSLog(@"%@",provinceDic);

获取海淀区人数 -> 海淀区 -> 区数组(北京) -> 北京市 -> 市数组

//获取城市数组
NSArray *cities = [provinceDic valueForKey:@"cityArr"];
        
//获取城市(北京)
NSDictionary *beiJing1 = [cities objectAtIndex:0];

NSDictionary *haiDian1 = [ares3 objectAtIndex:0];
